# Specification

## Summary
**Goal:** Make ProjectCard components more compact so more cards fit on screen at once.

**Planned changes:**
- Reduce the thumbnail/placeholder area height (from ~230px to ~120px or less)
- Tighten internal card padding to reduce overall card size
- Reduce the title font size while keeping it legible
- Cap description text at 1–2 lines with ellipsis overflow
- Reduce spacing between the Completion and Amazon Upload date rows
- Ensure at least 3–4 cards are visible per row on a standard desktop viewport
- Preserve status badge position and card click navigation behavior

**User-visible outcome:** Project cards are noticeably smaller and more compact, allowing more cards to be visible on screen at once without scrolling.
